Potion Letter Spacing

When adding letter-spacing to some text, take note of the fact that the spacing is added to the right of each letter, not just between letters.

This means that if you text-align: center text which has been letter-spaced, you may want to add add a negative right margin to remove the letter spacing added there, such that the text stays centered. Another similar option is to add a left padding.

This is a subtle thing, but it’s extremely good to know.
